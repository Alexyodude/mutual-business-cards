# mutual™ business cards

Static showcase of the mutual.solutions business card design system.

- **Live:** see GitHub Pages URL after deploy
- **Cards:** 1 shared front + 2 individual backs (Alex Jihoon Park, Yejun Jang)
- **Format:** 1050×600 PNG · 3.5″×2″ at 300 DPI · print-ready
- **Interaction:** click any card to flip between front and back (3D animation)
- **Download:** each side downloads as PNG via the action buttons

## Local preview

```
cd site
python -m http.server 8000
# open http://localhost:8000
```

## Source

Design rendered from HTML+CSS via Playwright at exact 1050×600 viewport. Source HTML lives in `../.canva-build/`.

Brand: dark navy `#0f172a` + cyan accent `#22d3ee` + slate. Inter (display) + JetBrains Mono (technical).
